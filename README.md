# RenderX
Simple Vulkan renderer

## Requirements
- Vulkan SDK
- CMake 3.20+

## Libraries
- GLFW (for example app)

## Parts of the renderer
- [x] VulkanInstance
- [x] VulkanDebugMessenger
- [x] VulkanSurface
- [x] VulkanPhysicalDevice
- [x] VulkanDevice
- [x] VulkanQueueFamily
- [x] VulkanSwapchainSupport
- [x] VulkanSwapchain
- [x] VulkanImageViews
